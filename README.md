# cocos-basic
cocos基础学习
```
assets：资源目录
    scene：场景
    material：材质 一般是给地图等等渲染
    prefab：预制-模型
    texture：预制-模型-质地
    script：脚本
    animation：动画
    font：字体
    audio：音频
    resources：其他资源

build：构建目录（在构建某平台后会生成该目录）

library：导入的资源目录

local：日志文件目录

profiles：编辑器配置

temp：临时文件目录

package.json：项目配置
```
